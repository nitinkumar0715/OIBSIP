# BMI Calculator

## Task 2 - Oasis Infobyte Internship

### 📌 Description
This is a simple Python BMI (Body Mass Index) Calculator that calculates BMI based on the user's height and weight and displays the corresponding health category.

### ✨ Features
- User-friendly interface
- Calculates BMI accurately
- Displays BMI category:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese
- Easy to use

### 🛠️ Technologies Used
- Python
- Tkinter (GUI)
